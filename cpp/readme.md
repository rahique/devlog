## **String**

`#include <string>` header file needs to be added to use string

### Getline

`getline()` is used to take string with sapce ` ` in it.

```cpp
    int main(){
        getline(cin, variable);
    }
```

### Size & Length

`size()` & `length()` are used to get the string length

```cpp
    int main(){
        std::string name = "rahqiue"
        std::cot << name.size()
        // 7
        std::cot << name.length()
        // 7
    }
```
